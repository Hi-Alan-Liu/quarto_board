## 🎮 遊戲介紹（What is Quarto）

**Quarto** 是一款兩人策略桌遊：

- 棋盤為 **4 × 4**
- 總共有 **16 顆棋子**
- 每顆棋子具有 **4 種屬性**（二元）：
  - 顏色：粉紅 / 粉藍
  - 高度：高 / 矮
  - 形狀：圓柱 / 方塊
  - 結構：中空 / 實心

🎯 **勝利條件**  
只要在同一條線（橫 / 直 / 斜）上，有 **4 顆棋子「共享任一屬性」**，即獲勝。

---

## 🕹️ 遊戲玩法（How to Play）

### 1️⃣ 選棋（你 → AI）
你先從右側棋子區 **選一顆棋子交給 AI**。

### 2️⃣ AI 放棋
AI 會把你選的棋子放到棋盤上。

### 3️⃣ AI 選棋（AI → 你）
AI 再選一顆棋子交給你（右側會被框起來）。

### 4️⃣ 你放棋
你必須將 AI 指定的棋子放到棋盤任一空格。

🔁 重複以上流程，直到：
- 有人獲勝 🎉
- 或棋子用完（平手 🤝）
